El dataset que elegi fue BMW Global Automotive Sales: https://www.kaggle.com/datasets/dmahajanbe23/bmw-global-automotive-sales

Lo que yo predeciria con este dataset es que modelo va a vender el proximo mes o año dependiendo de donde se venden. Usaria un modelo no supervisado, ya que hay muchos datos de prueba basados en años anteriores. Las features mas importantes me parecen primero la region, ya sea Europa, China, etc, despues el mes de venta que se cuenta y por ultimo el precio y la cantidad vendida.
